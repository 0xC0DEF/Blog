  2. BU DP할때 TD DP처럼 엣지케이스 처리하는 방법(0 base에서 -1참조할일 있을때 등 다양한곳에 활용가능)

    int& f(int i, int j){ static int d[N][M], a=inf, b=0,c=-1; if(i==-1)return a; if(j==-1) return b; return d[i][j];}

    f(i,j)=f(i-1,j)+f(i,j-1)
